# Message Board

Jonathan Thom

A simple message board application.

### Set Up

Deployed at [bk-message-board.herokuapp.com](http://bk-message-board.herokuapp.com/)

Or, you get it up and running through the following steps:

1. Clone it.
2. ```bundle install```
3. ```postgres```
4. ```rake db:setup```
5. ```rails s```

### Tech

Ruby on Rails

jQuery

And some cool gems like Devise, Bootstrap_Form, and Will_Paginate
